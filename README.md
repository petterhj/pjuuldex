# pokédex

Pokémon TCG collection database.


## Backend

```sh
$ cd backend/
$ pipenv shell
$ pipenv install
$ pipenv run api # Run development server locally
```

## Frontend

```sh
$ yarn install
$ yarn dev
$ yarn build
```
